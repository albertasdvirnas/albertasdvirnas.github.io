Research & Software

Projects

Open-source computational tools and research software for time series analysis, single-molecule imaging, and scientific computing.

🧬DOGMA

De novo assembly of densely labelled optical DNA maps using a matrix profile approach. Assembles DNA intensity barcodes into genome-scale maps without a reference sequence.

MATLABGenome AssemblyMatrix ProfileTime Series
📷EMCCD-PIA

Photon-optimal image analysis pipeline for EMCCD cameras used in single-molecule fluorescence microscopy experiments.

MATLABMicroscopyImage Analysis
🎯HPFL-ODM

High-precision fluorophore localization in nanochannel-based optical DNA mapping — algorithms for sub-diffraction position estimation.

MATLABDNA MappingLocalization
🔬SFW-Matlab

MATLAB implementation of the Sliding Frank–Wolfe algorithm for super-resolution microscopy and sparse signal reconstruction from ODM barcodes.

MATLABSuper-resolutionOptimisation
📊DTW-Matlab

MEX-compiled MATLAB interface to the Trillion DTW algorithm for ultra-fast dynamic time warping — a key tool for time series similarity and DNA barcode alignment.

MATLABC++Dynamic Time WarpingTime Series

Monte Carlo / molecular dynamics simulation of the Blume–Capel model using Wang–Landau sampling to compute the density of states.
